# Loops

## For Loop

`- Use `for` when looping over a known list or range.`

```bash
# As python
 
for var in list 
do 
# logic 
done 

{1..5} this is range from 1 to 5 

# Another syntax like c++

for ( i=0 ; i<3 ; i++))
do 
# logic 
done 
```
